Basico

Como ordenar tus ingresos y gastos

Empieza separando lo que entra, lo que sale y lo que inviertes. El objetivo no es registrar perfecto, sino detectar patrones que puedas mejorar cada mes.

  • Registra gastos grandes el mismo dia.
  • Revisa categorias al cierre de mes.
  • Compara tu flujo disponible contra tus metas.
Habitos

Fondo de emergencia antes de invertir

Un fondo de emergencia ayuda a no vender inversiones en mal momento. Puedes iniciar con una meta pequena y subirla poco a poco.

  • Primera meta: 1 mes de gastos.
  • Meta ideal: 3 a 6 meses.
  • Usa instrumentos liquidos y de bajo riesgo.
Inversion

Que revisar en tu portafolio

La app te ayuda a ver costo, valor actual y rendimiento. Para tomar mejores decisiones, revisa tambien diversificacion, horizonte y riesgo.

  • No concentres todo en un solo activo.
  • Define si inviertes a corto o largo plazo.
  • Rebalancea cuando una posicion pese demasiado.
Gastos

Senales de fuga de dinero

Muchas fugas no se sienten grandes por separado. La clave es revisar categorias repetidas: apps, comidas fuera, comisiones o compras impulsivas.

  • Busca gastos chicos que se repiten.
  • Pon limite mensual por categoria.
  • Cancela servicios que no usas.
GBM

Ideas para registrar inversiones

Captura ticker, titulos, precio promedio, precio actual manual y fecha de actualizacion. Si compras mas del mismo activo, puedes actualizar el promedio o crear una posicion nueva.

  • Actualiza precios una vez por semana o mes.
  • Guarda aportaciones como movimiento de inversion.
  • Exporta respaldo antes de cambios grandes.
Nota

Contenido educativo

Esta seccion es para aprender y organizarte mejor. No sustituye asesoria financiera personalizada ni toma en cuenta tu situacion completa.

  • Verifica fuentes antes de invertir.
  • Entiende riesgos, comisiones e impuestos.
  • No inviertas dinero que necesitas pronto.
Manual

Rutina mensual sugerida

Dedica 20 minutos al cierre de mes para revisar tus numeros. Con una rutina corta es mas facil mantener la app actualizada sin sentirla pesada.

  • Filtra el mes y revisa gastos por categoria.
  • Corrige movimientos sin descripcion.
  • Anota una accion concreta para el siguiente mes.
Precios

Como actualizar precios manuales

Consulta tu broker o una fuente confiable, captura el precio actual y registra la fecha. La app recalcula valor y rendimiento con ese dato.

  • Usa siempre la misma moneda de referencia.
  • No mezcles pesos y dolares en la misma posicion.
  • Actualiza todas tus posiciones el mismo dia.
Categorias

Como usar categorias sin complicarte

Las categorias sirven para decidir, no para hacer contabilidad perfecta. Si una categoria no te ayuda a tomar decisiones, mantenla simple.

  • Usa pocas categorias al inicio.
  • Separa necesidades, gustos e inversion.
  • Revisa solo las categorias que mas pesan.
Respaldo

Cuida tus datos locales

La app guarda datos en este navegador. Antes de limpiar cache, cambiar de equipo o publicar cambios, descarga un respaldo JSON desde Herramientas.

  • Haz respaldo una vez al mes.
  • Guarda el archivo en una carpeta privada.
  • Restaura el JSON si cambias de navegador.
Publicar

Primera version publica

Para compartirla, sube el archivo como index.html a GitHub Pages. Aclara que funciona de forma manual y que los datos se guardan en el navegador.

  • No necesita login ni base de datos.
  • No se conecta con bancos ni GBM.
  • Agrega tu enlace real de donacion.
Checklist

Antes de invertir mas

Una aportacion constante ayuda, pero primero revisa si tu flujo mensual la soporta y si ya tienes una base de seguridad.

  • Confirma que tu flujo disponible sea positivo.
  • Mantén separado tu fondo de emergencia.
  • Invierte segun horizonte y tolerancia al riesgo.
